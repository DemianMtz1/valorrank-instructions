Welcome to Valorrank

Learn how to use the skill Valorrank and discover the benefits...

Features

Calculate your stats

Using the skill, you can know the stats of any player in the current or the last match

MVP

You can know who is or were the MVP of the current or last match and his stats.

Recommendations and advice

The skill provided you a character recommendation that you can use for your next match and some advice that how can you play with them.

How to use

  1. Open Valorrank skill
  2. Go to this link "Building the link..."
  3. Type the user which you want to get the info.
  4. Listen to the options in your Alexa device and choose one.

Try saying

  • "Alexa, open Valorrank skill"
    This is the main way to start using the skill.
  • "Give me the mvp of the current ranked." or "Give me the mvp of the last ranked played."
    In this option, you can get the MVP for the current or last rank of the requested user on the website, after opening the skill.
  • "I want a recommendation" or "I want an advice"
    In this option, the skill can give you a recommendation for your next rank, after opening the skill.
  • "Calculate the stats"
    This option can give you the stats for the current or the last rank played, after opening the skill.
